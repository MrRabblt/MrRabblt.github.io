Typora的各个快捷键

1、标题

使用ctrl + 数字键就可以完成1~6级标题

2、下划线

ctrl + u

3、删除线

alt + shift +5

4、字体加粗

ctrl + b

5、字体倾斜

ctrl + i

6、无序列表

  • - + space
    • 无序嵌套

7、有序列表

  1. 数字键 + . + space

    1.1 嵌套列表

8、引用

> + space或者ctrl + shift + q

9、插入链接

ctrl+k

10、插入图片

直接拖动图片到编辑区或者使用快捷键ctrl + shift + i

11、代码块

12、文章跳转

ctrl + home 跳转到开头 ctrl + end跳转到文章末尾

13、选中英文单词

ctrl + d 或者 ctrl + shift + left/right 左右进行文本选中

14、按行选中

ctrl + l

15、快速搜索

ctrl + f 支持区分英文大小写和整个单词搜索，也支持中文搜索

16、替换

ctrl + h

17、快速生成表格

ctrl + t

18、快速打开文件

ctrl + p

19、toc快速生成目录

“[toc] + enter”

20、选中相同格式内容

ctrl + e

22、段落快捷键

ctrl + 0
